# 活动信息配置

当前框架可配置活动 ID、默认渠道号、百度统计分类名称。

**配置文件在 `./src/config/ActivityInformation.ts`**

|                                               | 对应字段             |
| --------------------------------------------- | -------------------- |
| 活动 ID                                       | ACT_ID               |
| 默认渠道号                                    | DEFAULT_CHANNEL      |
| [百度统计分类名称](./BaiduTrack#配置统计类别) | BAIDU_TRACK_CATEGORY |
